==============
Video Registry
==============

Scan directories to collect video files into a registry to build a webpage to
browse and for searching files.

Notes
*****

This is currently just a proof of concept.

************

* `Python`_>=3.5;
* `Click`_>=7.0,<8.0
* peewee==3.14.0
* Jinja2>=2.11,<3.0
* cherrypy>=18.6.0,<19.0.0
